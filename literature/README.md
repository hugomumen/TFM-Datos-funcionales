# Literatura y Referencias

Esta carpeta contiene material bibliográfico y referencias para el TFM.

## Estructura

```
literature/
├── papers/             # Artículos científicos relevantes
├── books/              # Referencias de libros (links o resúmenes)
├── bibliography/       # Archivos de bibliografía
│   ├── references.bib  # Archivo principal de BibTeX
│   └── notes.md        # Notas sobre referencias
├── reviews/            # Revisiones bibliográficas
└── summaries/          # Resúmenes de papers importantes
```

## Gestión de Referencias

### BibTeX
- Mantener un archivo `references.bib` principal
- Usar claves consistentes: `autor_año_titulo`
- Incluir DOI cuando esté disponible

### Organización de Papers
- Categorizar por tema o metodología
- Incluir resúmenes de papers clave
- Anotar relevancia para el TFM

### Revisiones de Literatura
- Estado del arte por temas
- Gaps identificados en la literatura
- Conexiones entre diferentes trabajos

## Herramientas Recomendadas

- Zotero/Mendeley para gestión de referencias
- JabRef para editar archivos .bib
- Exportar regularmente a BibTeX

## Buenas Prácticas

1. Leer y resumir papers importantes
2. Mantener notas sobre aplicabilidad al TFM
3. Actualizar bibliografía regularmente
4. Respaldar referencias en múltiples formatos
