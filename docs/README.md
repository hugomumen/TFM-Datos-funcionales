# Documentación

Esta carpeta contiene toda la documentación relacionada con el TFM.

## Contenido

- `thesis/` - Borradores y versiones de la tesis
- `notes/` - Notas de investigación y apuntes
- `presentations/` - Presentaciones y slides
- `meetings/` - Actas de reuniones con supervisores

## Estructura Sugerida

```
docs/
├── thesis/
│   ├── chapters/       # Capítulos individuales
│   ├── figures/        # Figuras para la tesis
│   ├── templates/      # Plantillas LaTeX
│   └── main.tex        # Documento principal
├── notes/
├── presentations/
└── meetings/
```

## Formato de Archivos

- Documentos principales en LaTeX (.tex)
- Notas en Markdown (.md)
- Presentaciones en formato preferido (.pptx, .tex con beamer)
